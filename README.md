# Mudra Sinhala SSL

Research repository for Sinhala sign-language related experiments, datasets, prompts, and model evaluation workflows.

## Repository Structure

- `Dataset/` - corpus files, names, token mappings, and supporting data.
- `Evaluation/` - evaluation scripts and result workbooks.
- `Prompts/` - prompt sets used in experiments.
- `Video Processing/` - video/dataset analysis scripts.
- `version-1/` - first round of model notebooks.
- `version-2/` - second round of model notebooks and result CSVs.

## Quick Start

1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install required packages used by the scripts/notebooks.
4. Run evaluation scripts from `Evaluation/` as needed.

Example:

```bash
python Evaluation/evaluate_all_models.py
```

## Notes

- Several experiments are notebook-based (`.ipynb`).
- Large data files and spreadsheets are included for analysis and reporting.
